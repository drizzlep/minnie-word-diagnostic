# 米妮单词诊断

手机优先的本地 PWA，用两次短测试诊断广州教科版（三年级起点）教材词表抽样的认义、听辨、拼写、词形和隔天保持情况。

## 教材来源

- 正式教材题均记录 `grade`、`semester`、`unit`、`sourceUrl`、`referencePage` 和核验日期。
- 三至六年级主核验源为[广州版小学英语三至六年级单词表](https://wenku.so.com/d/ed1b2ba3717bfcb5abc565e0934041a5)。
- 六年级下册补充核验源为[广州版小学英语六年级下册单词表](https://wenku.so.com/d/87b29fb5a0e89ff2065a277dcc1a8127)。
- 来源均为第三方汇总词表，不是出版社官方教材数据；题目记录了来源链接、核验日期和预览页，来源页面未来仍可能变化。
- 来源页码是汇总资料预览页，不是教材原书页码；应用结论仍是抽样诊断，不声称完整覆盖全部教材词汇。

## 使用

```bash
npm start
```

电脑浏览器打开 `http://localhost:4173`。首次完整打开后，应用和固定英式音频会缓存到本机，可离线继续。

手机不能通过普通 `http://<局域网IP>` 安装离线 PWA，因为 Service Worker 要求 HTTPS。手机正式使用时，需要将这个静态目录部署到 HTTPS 地址；在完成 HTTPS 部署和真机验收前，当前版本只保证电脑 `localhost` 的离线能力。

## 流程

- 第一天：3 道练习题、A 卷基础筛查、8 个现场新词学习与立即回忆。
- 第二天：至少间隔 18 小时后，先做新词延迟回忆，再完成长词定位。
- 全部完成后：生成家庭学习诊断、个人长度门槛、错误模式和可微调学习清单。
- 4–6 周后：导出首次报告，再从报告页开始等难度 B 卷复测。

应用只根据当前题库判断“广州教科版小学词汇抽测是否初步达标”，不代表完整教材覆盖、英语整体水平，也不提供同龄排名。不能把结果解释为广州小学全部词汇已经达标。

## 验证

```bash
npm run validate
```

该命令运行评分与报告测试，并检查题目 ID、长度分档和全部固定音频文件。

## 暑假训练（暂定教材版）

诊断完成前也可以从首页进入“暑假训练”。训练采用一套独立保存的、按新版沪教牛津七年级上册（广州地区）整理的高频共同核心词候选，不把它冒充为学校最终书单：

- 每天一轮 8 词，优先安排到期复习；没有到期词时补充新词，全部使用真实单词。
- 每个词先听英式发音，再看拆分/谐音线索，最后用应用内小写字母键盘拼写。
- 答对后按 1、2、4、7、14 天推进；答错不扣分，次日优先回来。
- 训练状态与诊断状态分开保存在同一台设备，可导出 `米妮暑假训练进度-YYYY-MM-DD.json`。
- 开学拿到教材后，只需按教材封面、版权页和单元词表做增删校准；已有训练记录保留，不静默清空。
